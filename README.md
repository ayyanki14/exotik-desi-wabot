# Exotik-Desi — WhatsApp Bot Demo
Phase 3: Interactive multilingual WhatsApp ordering bot demo (EN/TE/HI/TA).

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Import repo in Vercel dashboard
3. Framework: Vite
4. Build command: npm run build
5. Output directory: dist
